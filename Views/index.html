Definitions

HTML - HyperText Markup Language. HTML is a markup language used to make websites. It is often referred to as the structure of the website. The elements are represented by tags and the browser uses the tags to render the content.

CSS - Cascading Style Sheets. CSS is a language used to describe the presentation of HTML elements displayed on the screen. CSS controls all of the styles such as colors, fonts, alignment, etc.

JavaScript - JavaScript is a scripting language used for web development. It is used to enhance the HTML. JavaScript can make the webpage dynamic by including animations, reacting to events, accepting user inputs, etc.

HTTP - HyperText Transfer Protocol. HTTP is a protocol which defines how messages are formatted and transmitted. HTTP is a stateless protocol because every command is independent of the ones that came before it.

HTML5 - HTML5 is a W3C specification that defines the fifth revision of HTML. HTML5 has many new features including embedded graphics, audio, video, new elements to section the page like article, nav, source, and also new features which address web applications.

Dynamic HTML (DHTML) - Dynamic HyperText Markup Language. DHTML is a combination of web technologies including HTML, JS, CSS, and the DOM to enhance the user’s experience. This includes dynamic content which users can manipulate, dynamic positioning of elements, dynamic style allowing the user to change the page’s color, font, size, or content.
