jQuery

1. What you can do with jQuery?
jQuery is meant to help make the client-side scripting easier. It also simplifies creating events and animations on the webpage. One very useful feature is the "$(document).ready" function. The window.onload() has similar usage but it does not run until after all images are loaded and jQuery is able to bypass that.

2. How it can help you as web developer?
It makes it easier and requires less code to do things such as selecting elements and DOM manipulation based off of element names and attributes. It helps developers save time and space by providing convenient and efficient features.

3. If you think our earlier participant list app, in what kind of issues you could use jQuery?
You are able to use the add() function to add the new names to the person list. jQuery would also be able to simplify the code in regards to selecting elements and manipulating the CSS selectors related to those elements.

4. What are CDNs? What kind of benefits do you get if you use them?
CDN, or Content Delivery Network, is used as a means to efficiently spread services to a large amount of people. CDNs provide developers with availability to numerous services and in turn can increase their performance. CDNs are able to be pre-cached meaning if someone has visited the site before the file will already be downloaded. Many of the CDNs have data centers across the world so if an international user is trying to acess your site a CDN may improve the speed. A CDN can distribute the load, save bandwidth, boost performance, and lower your hosting costs.
