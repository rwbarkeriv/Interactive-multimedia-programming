Responsive Web Design

What is the purpose of the RWD? Responsive Web Design is utilized to dynamically adjust the content of a web page to fit all screen sizes.

Why it is used? It is used so that the content can be easily viewed on all platforms. If your page is not responsive and you try to visit the webpage on a small screen like a cell phone the content may be difficult to view and require horizontal scrolling to see all of the content. So instead of creating several websites that are formatted to a specific screen size we can use RWD and have one page that fits all screen sizes.

How things are done before RWD? Before RWD developers had to create several different webpages to fit the different screen sizes. This can also be seen when developers had to create solely mobile webpages so that the users on a cellphone could read all of the content on the page.

What is the difference? The difference is that we now can make one webpage that looks good on all screen sizes. Also modern RWD technologies have made it much easier to layout the page.

How css media queries can help you building responsive web sites? CSS media queries are a great tool that can help when building responsive pages. They adjust the content based on the viewport's height and width in order to show all of the content in an aesthetic manner. Media queries have breakpoints that are set so that when the screen gets to a certain width or height the content will dynamically adjust. Many css frameworks and libraries have built-in media queries that can help create responsive sites.

Why use em (font) based sizing for margins? The em sizing is a scalable unit that is used to allow the user to adjust the font-size in the browser. 1em is the equivalent to whatever the page's default size is or has been set to. Default browser size is 16px, this would be equal to 1em. Using em in margins will mean that it is related to the pages font-size. So whether the user has large or small text the margins will scale accordingly.

Article is already 7 years old, do we have any new technology which can be used to achieve same type of UI responsiveness as described in the article. Many of the new CSS libraries and frameworks incorporate new RWD technologies such as the grid system and Flexbox. Bootstrap for instance is based on a 12 column grid system and also incorporates CSS Flexbox. CSS Grid helps to properly layout the content of the website contains media queries that help make it responsive. CSS Flexbox makes it easier to layout a page without having to use floats or positioning.
